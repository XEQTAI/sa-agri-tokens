## üìñ **How to Start Trading**

### Step 1: Install Freighter Wallet
1. Visit [freighter.app](https://www.freighter.app/)
2. Install browser extension
3. Create or import Stellar account

### Step 2: Switch to Testnet
1. Click Freighter extension
2. Go to Settings
3. Select **"Testnet"** network
4. Confirm switch

### Step 3: Fund Your Account
1. Copy your public key from Freighter
2. Visit [Stellar Laboratory](https://laboratory.stellar.org/#account-creator?network=test)
3. Paste your public key
4. Click "Get Test Network Lumens"
5. You'll receive 10,000 XLM (testnet)

### Step 4: Connect to Exchange
1. Visit [Your Exchange](https://xeqtai.github.io/sa-agri-tokens/exchange.html)
2. Click "Connect Wallet"
3. Approve Freighter popup
4. You're connected! üéâ

### Step 5: Add Trustlines
Before trading, add trustlines for each token:
1. Click "‚ûï Add Trustline" for MAIZE
2. Approve in Freighter
3. Repeat for WHEAT and SOYSA
4. Each trustline costs ~0.00001 XLM

### Step 6: Start Trading!
1. Click "üõí Buy MAIZE" (or any token)
2. Enter amount (e.g., 10 MAIZE)
3. Enter price (e.g., 1.0 XLM per MAIZE)
4. Click "Buy MAIZE"
5. Approve transaction in Freighter
6. Done! Your order is on the Stellar DEX

## üêõ **Troubleshooting**

### "Connect Wallet" button doesn't work
- **Solution:** Install Freighter from [freighter.app](https://www.freighter.app/)
- Make sure Freighter is unlocked

### "Please switch to TESTNET network"
- **Solution:** Open Freighter ‚Üí Settings ‚Üí Select "Testnet"

### "Account not found"
- **Solution:** Fund your account from [Stellar Laboratory](https://laboratory.stellar.org/#account-creator?network=test)
- Need minimum 1 XLM to exist on network

### "Failed to add trustline"
- **Solution:** Check you have XLM balance
- Each trustline requires minimum account balance

### Trade not filling
- **Normal!** Orders on DEX wait for matching orders
- Try adjusting your price
- Check on [Stellar Expert](https://stellar.expert/explorer/testnet)
